Weekly budget management email

Subject: Acme Hydraulics - Google Ads Weekly Report

Prompt OptiMate saves/runs weekly

Call get_weekly_metric_table with metrics=["spend","conversions","cpa"], weeks=4 by default, or the number of weeks requested by the user, and endDate set to the previous Sunday in agency time so the rows cover completed Monday-Sunday weeks.

Then call get_budget_management_email with mode="this_month" to get the canonical Budget Management HTML.

Write 1 to 2 plain Gmail-safe sentences above the report using one positive data point from the weekly rows. Build the email as: greeting, positive weekly intro, weekly table HTML verbatim, then the budget HTML verbatim.

Use subject: [Client Name] - Google Ads Weekly Report.

Summary sentence logic

  • Use only values returned by the weekly tool: spend, conversions, CPA, and any available comparison rows/context.
  • Prefer a clearly positive fact in this order: conversions up/strong, CPA improved/low, CPC efficient if available, spend controlled/on pace.
  • If only one week is available, avoid fake comparisons. Say the actual positive point, e.g. “CPA came in at $96 with 57 conversions.”
  • Keep it client-safe: no internal caveats, no unsupported claims, no coloured custom callout.

Hey team,

Great week across Google Ads: conversions increased to 57 while CPA improved to $96. Spend stayed controlled, keeping the account just under budget and giving us a strong base for the rest of the month.

Create the monthly budget email with dashboard components: keyword_relevancy, cpa_trend, quality_score, top_converters.

Prompt OptiMate uses for monthly/MTD

Call get_dashboard_email_components with components in the selected order. For keyword_relevancy, cpa_trend, and quality_score, use the canonical graph renderer so the email graph style stays consistent.

Then call get_budget_management_email with mode="this_month" to get the canonical Budget Management HTML.

If a short client-facing summary is requested, base it on tool data from the current conversation or a fresh read tool. Build the email as: greeting, summary paragraph if requested, selected component HTML from get_dashboard_email_components.data.html, then the canonical Budget Management HTML unchanged.

Use the monthly Gmail subject: [Client Name] - Google Ads Monthly Report - [Month Year].

Monthly summary logic

  • Only add these sections when selected/requested on-call: Keyword Relevancy % graph, CPA Trend graph, Quality Score graph, Top converters.
  • Keep all graph sections visually consistent, with CPA using its own orange trend colour.
  • Keep Top converters as a compact table.
  • Keep the canonical Budget Management HTML unchanged below the selected sections.

Hey team,

Excluding display, May finished just under the $6,000 budget at $5,474.10, while performance improved from April with conversions up from 30 to 57, CPA down from $166 to $96, and CPC down from $4.50 to $3.39.
