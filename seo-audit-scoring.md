# SEO Audit Scoring Breakdown

Each category is scored 0–10 per page. The overall score is a weighted average of all category scores, rounded and clamped to 1–10.

| # | Category | Weight | What It Scores |
|---|---|---|---|
| 1 | Core Web Vitals | 12% | Google PageSpeed performance score for mobile (60% weight) and desktop (40% weight) — LCP, CLS, TBT |
| 2 | Structured Data | 10% | JSON-LD schema markup presence and correctness per page type (Organization, BreadcrumbList, FAQPage, Service, Article) |
| 3 | Navigation & UX | 10% | Semantic nav element, footer links, viewport meta, breadcrumbs, 3-8 nav items, logo linking home, header CTA |
| 4 | Service Coverage | 10% | Services section on homepage, links to individual service pages, USP/differentiator content, service-specific CTAs |
| 5 | Meta Data | 9% | Title tag (length, uniqueness), meta description (120-160 chars), canonical tag, Open Graph tags, Twitter card |
| 6 | E-E-A-T | 9% | Trust signals: testimonials/reviews, credentials/certifications, results/case studies, author info, social links, contact info |
| 7 | Heading Structure | 8% | Single H1 per page, valid hierarchy (H1>H2>H3), heading length under 80 chars, no CTA phrases in headings |
| 8 | Internal Linking | 8% | In-content contextual links (not just nav/footer), descriptive anchor text, distinct link targets across the page |
| 9 | URL Structure | 8% | Clean URLs (no query params), lowercase, hyphen-separated, descriptive paths, canonical tag, breadcrumb nav + schema |
| 10 | Content Structure | 8% | Word count thresholds by page type, paragraph scannability, lists/emphasis usage, H1-to-title alignment |
| 11 | Image Optimization | 5% | Alt text coverage, descriptive alt text, modern formats (WebP/AVIF), explicit dimensions, lazy loading |
| 12 | FAQ Implementation | 3% | FAQ section presence (accordion/details), 2-8 questions, FAQPage JSON-LD schema for rich results |
