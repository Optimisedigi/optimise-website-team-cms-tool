# Content CMS - Team Guide

Welcome to the Content CMS! This guide will teach you how to create SEO-optimized blog posts for our client websites.

---

## Getting Started

### Logging In

1. Go to `cms.yourcompany.com/admin` (or `localhost:3001/admin` for local development)
2. Enter your email and password
3. You're in!

---

## Creating a Blog Post

### Step 1: Click "Blog Posts" in the sidebar

### Step 2: Click "Create New"

### Step 3: Select the Client

At the top right, you'll see a dropdown to select which client this blog post is for. **Always select the correct client first.**

### Step 4: Fill in the Content Tab

#### Title (H1)
This is the main headline of your blog post. Make it:
- **Intent-led**: Describe what the reader will learn
- **Specific**: Include who it's for

**Good examples:**
- "How to Speed Up Your WordPress Site in 2024"
- "5 SEO Mistakes Small Businesses Make (And How to Fix Them)"

**Bad examples:**
- "Website Tips" (too vague)
- "Our Thoughts on SEO" (not intent-led)

#### Excerpt
Write a 1-2 sentence summary. This appears:
- In search results (Google)
- On the blog listing page
- When shared on social media

Keep it under 160 characters.

#### Content (The Main Body)
This is where you write your blog post. **Use proper heading structure for SEO!**

---

## Heading Structure for SEO (IMPORTANT!)

When writing your blog content, use the heading buttons in the editor toolbar.

### The Rules

| Heading | When to Use | Example |
|---------|-------------|---------|
| **H2** | Main sections of your article | "Why Website Speed Matters" |
| **H3** | Subsections within an H2 | "1. Use a CDN" |
| **H4** | Rarely needed, for deep subsections | "Configuring Cloudflare" |

### Never Do This
- ❌ Don't use H1 in the content (the title is already H1)
- ❌ Don't skip levels (H2 → H4 without H3)
- ❌ Don't use headings for styling (use bold instead)

### Good Structure Example

```
Title: How to Improve Your Website Speed (this is H1 - automatic)

[Introduction paragraph - no heading]

H2: Why Website Speed Matters
   [Paragraph explaining importance]

   H3: Impact on SEO
      [Details about SEO impact]

   H3: Impact on Conversions
      [Details about conversions]

H2: 5 Ways to Speed Up Your Site

   H3: 1. Optimize Your Images
      [Explanation]

   H3: 2. Use Browser Caching
      [Explanation]

   H3: 3. Minimize JavaScript
      [Explanation]

H2: Conclusion
   [Summary and call to action]
```

### How to Add Headings in the Editor

1. Highlight the text you want to make a heading
2. Click the paragraph dropdown (it might show "Normal" or "Paragraph")
3. Select "Heading 2" for main sections, "Heading 3" for subsections

## Adding Images

### Featured Image
- Upload in the "Media & Display" tab
- This becomes the thumbnail on the blog listing and social shares
- **Always add alt text** describing the image

### Images Within Content
1. Click the image icon in the editor
2. Upload your image
3. Add alt text (describe what's in the image)

**Good alt text:** "Developer reviewing code on a MacBook in a modern office"
**Bad alt text:** "image1" or "photo"

---

## SEO & Meta Tab

### Slug
The URL for your post. Use:
- Lowercase letters
- Hyphens between words
- No spaces or special characters

**Example:** `how-to-speed-up-wordpress`

### Meta Title (Optional)
If you want a different title for search results. Usually you can leave this blank.

### Meta Description (Optional)
If you want a different description for search results. Usually the excerpt is fine.

---

## Tags

Add relevant tags to help with filtering and SEO:
- Use existing tags when possible
- Keep tags consistent across posts
- 2-5 tags per post is ideal

## Publishing

### Status Options

| Status | What It Means |
|--------|---------------|
| Draft | Work in progress, only visible to CMS users |
| Ready for Review | Finished, waiting for approval |
| Published | Live on the website |

### Before Publishing Checklist

- [ ] Title is clear and intent-led
- [ ] Excerpt is under 160 characters
- [ ] Content uses proper H2/H3 structure
- [ ] Featured image is uploaded with alt text
- [ ] Slug is clean and readable
- [ ] Tags are added
- [ ] Correct client is selected
- [ ] Author name is correct
- [ ] Publication date is set

---

## Quick Tips

1. **Write for humans first, SEO second** - If it reads naturally, it's usually good for SEO
2. **One topic per post** - Don't try to cover everything
3. **Use short paragraphs** - 2-4 sentences max
4. **Add internal links** - Link to other relevant blog posts when appropriate
5. **Proofread before publishing** - Use the Preview feature
